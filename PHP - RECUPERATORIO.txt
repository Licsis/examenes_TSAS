Marcos tiene la costumbre de siempre guardar los tickets de compras para tener un control de sus finanzas, lamentablemente trabaja con lapiz, papel y esporádicamente con hojas de cálculos, por ello a contratado tus servicios para que en esta primer entrega le presentes un sistema más robusto y el pueda almacenar en una base de datos el nombre del lugar donde compró, en nombre del producto, el precio unitario y la cantidad. El administrador de la base de datos ya diseñó la base de datos inicial, solo falta programar las funciones del mismo con php, para ello:

1 Crea una base de datos local llamada "tickets" usando xampp (el cotejamiento de la base de datos debe ser utf8_spanish_ci).
2 importar el archivo tickets.sql con los datos de la base de datos a xampp, la cual se encuentra en el siguiente repositorio: https://github.com/Licsis/examenes_TSAS.git
3 mediante archivos de html, css y php debes desarrollar un sistema para cargar, borrar o modificar registros de la tabla "boleto_compra".
4 la estructura del sistema de archivos es:

index.html -> visualizador.php -> abm_boleto/Aboleto1.html 
			       -> abm_boleto/Aboleto2.php  
			       -> abm_boleto/Bboleto.php
			       -> abm_boleto/Mboleto1.php 
			       -> abm_boleto/Mboleto2.php 

En donde el archivo index.html conectará con un botón a visualizador.php y este con los demás archivos del ABM, los cuales estarán en una carpeta llamada "abm_boleto".

El recuperatorio estará aprobado con el sistema ABM funcionando con la creación, modificación y eliminación de los productos de la base de datos local.
